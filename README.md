# MeWay

## Getting Started

> **Note**: This repo uses `pnpm` instead of `npm` or `yarn`

```sh
# 1. Clone this repository

# 2. Start from the root directory of the repo
cd meway

# 3. Install the dependencies
pnpm install

# 4. Install Husky Hooks (!!! DO NOT SKIP THIS !!!)
pnpm hooks

# 5. Build the monorepo
pnpm build
```
